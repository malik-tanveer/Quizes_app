## Usage

1. Open the app in your browser.
2. Fill in the registration form:
   - Name
   - Father's Name
   - GR Number
   - Email
   - Shift (Morning / Evening)
3. Click **Submit** to start the quiz.
4. Answer each question and click **Submit** for each one.
5. Click **Next** to go to the next question.
6. Once completed, your results will be displayed with feedback.

---

## Customization

- **Change Name in App:** Update `userName` or default placeholders in `index.html`.
- **Update Year:** Modify the footer copyright.
- **Add Questions:** Add new objects to the `quizData` array in `web.js`.
- **Theme Adjustments:** Edit `style.css` for colors, fonts, or dark/light mode preferences.
